# Example App Fonts

The example gallery bundles these open-source fonts:

- Fraunces, licensed under the SIL Open Font License 1.1.
- Inter Tight, licensed under the SIL Open Font License 1.1.
- JetBrains Mono, licensed under the SIL Open Font License 1.1.

These fonts are used only by the example app and are not runtime dependencies of
the `math_curve_loaders` package.
